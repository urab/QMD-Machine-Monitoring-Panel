# QMD Machine Monitoring Panel

Industrial machine monitoring system for injection molding machines.

## Features

- Real-time monitoring of 12 machines
- Ethernet Web Dashboard
- Alarm management
- Smart Mute logic
- Water temperature monitoring
- Built-in thermostat
- DS3231 Real Time Clock
- REST API
- Arduino Mega 2560
- Ethernet W5100

## Hardware

- Arduino Mega 2560
- W5100 Ethernet Shield
- DS3231 RTC
- DS18B20
- 36 opto-isolated digital inputs
- Alarm relay
- Thermostat relay
